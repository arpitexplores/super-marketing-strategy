## Why SUPER Skills For AI Vibe Coding

Super Marketing Strategy helps AI coding agents define the market logic before execution. Use it for ICP, positioning, messaging, product marketing context, content strategy, launch planning, and growth hypotheses that make campaigns sharper.

These skills are designed for **AI vibe coding**: fast, agent-assisted building where the AI needs strong domain context, practical workflows, guardrails, and implementation-ready outputs.

**SEO and discovery keywords:** marketing strategy, positioning, ICP, product marketing, content strategy, go-to-market, GTM, growth strategy, AI vibe coding, agent skills.

## Install

Copy this folder into your agent's skills directory, then restart or reload the agent.

```bash
cp -R super-marketing-strategy ~/.your-agent/skills/
```

Use it by name:

```text
Use $super-marketing-strategy to help with this request.
```
